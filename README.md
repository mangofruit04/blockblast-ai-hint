# 🎮 Block Blast AI Hint

A Block Blast game clone developed in **Python** using **Pygame**, featuring an **AI-powered Hint System** that recommends the best possible move for the player.

## ✨ Features

- 🎲 Classic Block Blast gameplay
- 🤖 AI-powered Hint System
- 🧠 Uses a Best-First Search heuristic to recommend optimal block placement
- 🏆 Score tracking
- 🎨 Clean and colorful user interface
- 🕹️ Drag-and-drop block placement

## 🛠️ Technologies Used

- Python
- Pygame
- Heap Queue (`heapq`)
- Best-First Search Algorithm

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/mangofruit04/blockblast-ai-hint.git
```

2. Install the required package

```bash
pip install pygame
```

3. Run the game

```bash
python main.py
```

## 🎯 Controls

- Drag and drop blocks onto the board.
- Press **H** to display the AI Hint for the best move.
